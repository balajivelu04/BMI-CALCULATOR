# BMI Calculator

A simple Java application that calculates Body Mass Index (BMI) using Swing for the user interface.

## Features

- Input weight in kilograms and height in meters
- Calculates BMI and provides health status
- Categories: Underweight, Normal, Overweight, Obese
- Error handling for invalid inputs

## Prerequisites

- Java 8 or higher
- Maven

## How to Run

1. Clone the repository
2. Navigate to the project directory
3. Run `mvn clean compile`
4. Run `mvn exec:java -Dexec.mainClass="com.leo.App"`

## Usage

Run the application and enter your weight and height when prompted. The BMI and health status will be displayed.
